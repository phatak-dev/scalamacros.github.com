Polishing of the reflection API has resulted in drastic changes both on conceptual and technical levels. Therefore the information provided in the documents below is now partially obsolete. For a definitive reference, refer to the slides provided above, pre-release scaladocs for scala.reflect.api and scala.reflect.macros or reflection guide at docs.scala-lang.org.

The revised macro SIP introduces significant changes to our original ideas: the syntax is now different, some features are postponed until the next release, and instead of quasiquotes we now use reify, another macro that provides comparable functionality.

However these documents linked below still capture the essence of our project - bringing compile-time metaprogramming to Scala, so they are a good and useful read anyway.
