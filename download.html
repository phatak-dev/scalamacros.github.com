Macros ship with Scala starting from 2.10.0 M3. Use download links at Scala home page, or refer to an appropriate Scala version in Maven or SBT to get a macro-enabled Scala compiler.

If you feel like experimenting, you can clone our bleeding edge code repository at GitHub is {{ site.github.url }}. The develop branch is the most stable, other branches host experimental features. Building the repository should be simple if you follow the instructions in the readme, setting up an IDE is more complex and is described here.

We also warmly welcome contributors. If you feel like working on project Kepler, we will be very glad to hear from you at {{ site.email }}.
