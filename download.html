Macros ship with Scala starting from 2.10.0 M3. Use download links at Scala home page, or refer to an appropriate Scala version (2.10.0-M6 or 2.10.0-SNAPSHOT) in Maven or SBT.

Public compiler API (aka reflection API) has undergone major rehash in M4, so examples and documentation from M3 will most likely not work for M4 and vice versa. This site only covers the most recent developments and doesn't provide info about macros and reflection in M3.

If you feel like experimenting, you can clone our bleeding edge code repository at GitHub is {{ site.github.url }}. The develop branch is the most stable, other branches host experimental features. Building the repository should be as simple as running ant, setting up an IDE is described here.
