Metaprogramming in Scala 2.10. Overview of the reflection API: briefly explains core data structures and gives a few recipes on discovering relevant information using hidden switches of compiler and REPL. The guide has been brought up to speed with 2.10.0-RC1

Pre-release scaladoc for scala.reflect.api. Outlines the data structures and factories that are used inside the compiler and their public API. Trees, symbols, types, annotations, positions, and more. The link won't work until 2.10.0-RC1 is released.

Pre-release scaladoc for scala.reflect.macros. Explains what can be done from within a macro, ranging from printing custom error messages to performing implicit searches.The link won't work until 2.10.0-RC1 is released.

Reflection guide at docs.scala-lang.org. Intended to be something like a collection guide for Scala reflection. This guide will subsume the overview linked above by the time 2.10.0 hits release.The link won't work until 2.10.0-RC1 is released.
