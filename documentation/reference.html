scala/reflect/makro/Context.scala: Sources of macro context API: explains what can be done from within a macro, ranging from printing custom error messages to performing implicit searches.

scala/reflect/base and scala/reflect/api: Sources of reflection API: exposes data structures and factories that are used inside the compiler. Trees, symbols, types, annotations, positions, and more.

Metaprogramming in Scala 2.10: Overview of reflection API: briefly explains core data structures and gives a few recipes on discovering relevant information using hidden switches of compiler and REPL. Since the release of 2.10.0-M4 some of that information is obsolete. The guide has been updated with the information about the changes introduced in 2.10.0-M4 and 2.10.0-M5.

Scala 2.10.0 Milestone 4: Summary of the changes introduced to the public compiler API (aka reflection API) in M4. The linked thread also contains migration instructions for 2.10.0-M3 users.

Pre-SIP: Scala Reflection: A write-up on the new design of the reflection API. For now it's the definitive reference on reflection, will eventually end up as a Scala improvement proposal.
